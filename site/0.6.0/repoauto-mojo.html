proxy:repoauto

Full name:

org.duguo.maven.plugins:maven-proxy-plugin:0.6.0:repoauto

Description:

Running as maven repository proxy with auto detected configuration based on current maven runtime environment.

Attributes:

Optional Parameters

Name Type Since Description
addProjectRemoteReposAsIgnored boolean - add project remote repos as ignored repositories, check first before go to proxy repo.
Default value is: true.
cetralMirrorRepoUrl String - Central repository uk mirror url
Default value is: http://uk.maven.org/maven2.
cetralRepoUrl String - Central repository url
Default value is: http://repo1.maven.org/maven2.
cetralRepoValidationResource String - The file will be checked to indicate the central repo is accessible
Default value is: /org/apache/maven/plugins/maven-metadata.xml.
checksumExtensions String[] - Checksum file extensions, defaults include .md5 and .sha1
doNotCopyLocalRepoFile boolean - Releases repository storage base
Default value is: false.
hostname String - Host name of the proxy server
Default value is: localhost.
jarArtifactAdditionalClassifier String[] - Checksum file extensions, defaults include sources and javadoc
javaNetRepoUrl String - java.net repository url
Default value is: http://download.java.net/maven/2.
pluginsRepositoryBase String - Plugins repository storage base
Default value is: ${basedir}/target/repos/plugins.
pluginsRepositoryPrefix String - Plugins repository prefix
Default value is: /plugins.
port int - Listen port of the proxy server
Default value is: 8080.
releasesRepositoryBase String - Releases repository storage base
Default value is: ${basedir}/target/repos/releases.
releasesRepositoryPrefix String - Releases repository prefix
Default value is: /releases.
tempDir File - Temporary folder to hold index file for folder list request
Default value is: ${basedir}/target/tmp.
useMirrorForCentralServer boolean - Use mirror for central server
Default value is: false.
virtualRepositories List - Virtual repository exposed by this proxy.
