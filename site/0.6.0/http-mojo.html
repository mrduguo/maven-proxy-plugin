proxy:http

Full name:

org.duguo.maven.plugins:maven-proxy-plugin:0.6.0:http

Description:

Running as general http proxy server to access resources via wagon.

Attributes:

Optional Parameters

Name Type Since Description
doNotCopyLocalRepoFile boolean - Releases repository storage base
Default value is: false.
hostname String - Host name of the proxy server
Default value is: localhost.
port int - Listen port of the proxy server
Default value is: 8080.
tempDir File - Temporary folder to hold index file for folder list request
Default value is: ${basedir}/target/tmp.
virtualRepositories List - Virtual repository exposed by this proxy.

Parameter Details

doNotCopyLocalRepoFile:

Releases repository storage base
  • Type: boolean
  • Required: No
  • Expression: ${doNotCopyLocalRepoFile}
  • Default: false

hostname:

Host name of the proxy server
  • Type: java.lang.String
  • Required: No
  • Expression: ${hostname}
  • Default: localhost

port:

Listen port of the proxy server
  • Type: int
  • Required: No
  • Expression: ${port}
  • Default: 8080

tempDir:

Temporary folder to hold index file for folder list request
  • Type: java.io.File
  • Required: No
  • Expression: ${tempDir}
  • Default: ${basedir}/target/tmp

virtualRepositories:

Virtual repository exposed by this proxy.
  • Type: java.util.List
  • Required: No
  • Expression: ${virtualRepositories}
